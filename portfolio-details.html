ExitMap

A BASE jumping community.

www.exit-map.com

  • Full-stack web application that allows BASE jumpers to input and view jumps from around the world.

  • Stack: React, Typescript, NodeJS/Express, PostgreSQL, Redis, Amazon S3
  • APIs used: Google Maps API, Google Geocoding API, reCAPTCHA, Nodemailer API, AWS SDK
  • Features: User authentication/verification, session management, user image upload/optimization, RDBMS.

The worlds largest BASE jumping information community.

ExitMap is THE community for BASE jumpers to get access to information to promote safety and cooperation between jumpers from around the world. ExitMap offers exclusive access only to verified BASE jumpers so that the community can remain safe and out of the spotlight, as many jumpers prefer. ExitMap retains the true Wild West spirt of BASE jumping while promoting an innovative change to BASE culture.
